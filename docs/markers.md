# Markers

Drop pins on the chart, name them, and navigate to them with live bearing and distance.

## Features

- **Drop a marker** — Tap the "Marker" button in the location sheet to pin any location on the chart.
- **Name and annotate** — Give each marker a name, icon, color, and optional notes. Colors and icons help organize markers by purpose (e.g. red for hazards, green for anchorages).
- **Map display** — All markers render as a native circle layer on the chart. The selected marker shows as an interactive annotation.
- **Live bearing and distance** — The marker detail view shows continuously updated bearing and distance from the vessel's current position.
- **Drag to reposition** — Select a marker on the chart and drag it to a new location.
- **Manage** — Browse all markers in a searchable list, sorted by name, date created, or proximity. Rename or delete from the detail view.

## Dropping a marker

From the location sheet (tap anywhere on the chart), tap the "Marker" button. This opens a creation form pre-filled with the selected coordinates, with fields for name, notes, icon, and color.

## Marker detail

Tap a marker on the chart or in the list to open its detail sheet:

- Name and coordinates
- Live bearing and distance from current GPS position
- Notes
- Edit (name, notes, color, icon) and delete actions

## Export and sharing

From the marker detail sheet, the Share menu offers:

- **Export GPX** — Save the marker as a standard GPX 1.1 `<wpt>` element and share via the iOS share sheet (AirDrop, Files, other apps).
- **Open in...** — Open the marker's location in other mapping apps (Apple Maps, Google Maps, etc.).
